# Recommended Mods for Quest

Mods are installed through **MBF Launcher** after you've modded your game. See the [Quest Modding guide](/quest) if you haven't set that up yet.

---

## 1.37.0

**Main Pack** is recommended for most players — it includes the most common mods in one download:

**[Download Main Pack](/modPacks/mainModBundle.qmod)**

| Mod | What it does |
|---|---|
| **BeatLeader** | Global leaderboards with replays and ranked play. |
| **BeatTogether** | Multiplayer with modded players across Quest and PC. |
| **BeatSaber Plus** | Real-time multiplayer and co-op with friends. |
| **Better Song Search** | In-game song browser powered by BeatSaver — search and download custom maps without leaving the headset. |
| **Chroma** | Extended RGB lighting effects for maps that use them. |
| **Clock UI** | Shows a clock in the main menu so you know what time it is without taking the headset off. |
| **Crash Reporter** | Strongly recommended — captures crash logs and lets support staff diagnose game crashes quickly. |
| **Graphics Tweaks** | Fine-tune render quality, bloom, and shockwave effects. |
| **Score Visualizer** | Shows colour-coded score popups on every hit. |
| **Intro Skip** | Skips the Beat Saber intro video so you get to the menu faster. |
| **Mapping Extensions** | Required to play maps that use extra lanes or non-standard note positions. |
| **Noodle Extensions** | Required to play maps with custom note movements and choreography. |
| **No Promo** | Removes promotional banners and pop-ups from the main menu. |
| **Counters Plus** | Adds configurable HUD counters (score, combo, miss count, time, etc.) during gameplay. |
| **Custom Hit Sounds** | Replace the default note hit and miss sounds with custom audio files. |
| **Recently Played** | Adds a "Recently Played" category to the song list. |
| **Score Percentage** | Shows your current score as a percentage in real time. |
| **Slice Details** | Displays cut angle and accuracy info after each note hit. |
| **Song Chart Visualizer** | Shows a note density chart above the song list preview. |

## Vivify on Quest

::: warning Not available yet
**Vivify is currently in development for Quest and has no release date.**

Vivify is a next-generation environment and visual effects framework used by newer PC maps. A Quest port is being worked on, but it has not been released and there is no ETA.

Maps that require Vivify will **not load correctly** on Quest until it is released. If a map's BeatSaver page lists Vivify as a requirement, it is PC-only for now.
:::
