# Recommended Mods for PC

Mods are installed through **BSManager** → Mods tab. See the [PC Modding guide](/pc) if you haven't set that up yet.

---

## 1.40.7

Current recommended moddable version. All mods below are available through BSManager.

### Essential

| Mod | What it does |
|---|---|
| **SongCore** | Required by almost every other mod — loads custom songs and handles the custom levels menu. Install this first. |
| **BeatSaverDownloader** | Adds a full in-game custom song browser powered by BeatSaver. Search, preview, and download without leaving the headset. |
| **BeatSaverVoting** | Lets you upvote or downvote maps while in-game. |
| **PlaylistManager** | Create and manage playlists of custom songs in-game. |
| **BetterSongList** | Adds filtering, sorting, and search improvements to the song list. |

### Leaderboards & Stats

| Mod | What it does |
|---|---|
| **ScoreSaber** | Global leaderboards and ranked play. Required for competitive scoring. |
| **BeatLeader** | Alternative leaderboard with replays, clan system, and detailed stats. |
| **HitScoreVisualizer** | Shows colour-coded score popups on each hit so you can see exactly how well you're cutting. |
| **SongChartVisualizer** | Displays a note density chart during gameplay so you can see what's coming. |

### Gameplay & QoL

| Mod | What it does |
|---|---|
| **BetterPause** | Improves the pause menu — shows your current score, lets you restart or quit cleanly. |
| **TrickSaber** | Lets you spin and throw your sabers for style. |
| **Camera2** | Powerful camera system for streamers — custom positions, movement scripts, and multi-camera setups. |

---

## Mods for Custom Map Formats

These are required to play maps that use advanced lighting or movement effects. Install them if a map's BeatSaver page lists them as requirements.

| Mod | What it does |
|---|---|
| **Chroma** | Extended RGB lighting effects and environment colouring for maps. |
| **NoodleExtensions** | Enables custom note movements, rotations, and complex map choreography. |
| **MappingExtensions** | Adds extra precision lanes and note positions beyond the standard grid. |
| **Vivify** | Next-generation environment and visual effects framework used by newer maps. |
| **AudioLink** | Syncs environment lighting and effects to the game's audio in real time. |

---

## Where to Find More Mods

- **[BeatMods](https://beatmods.com)** — the official mod repository. BSManager pulls from here automatically.
- **[BeatSaver](https://beatsaver.com)** — custom song maps (not mods).
- **[ModelSaber](https://modelsaber.com)** — custom sabers, avatars, and platforms.
